1.1- ¿Que significa CSS?

CSS significa (Cascading Style Sheets) u Hojas de estilo en cascada. Es decir, te permite aplicar estilos de manera selectiva a elementos en documentos HTML.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

CSS, o Hojas de Estilo en Cascada, sirve para dar estilo a los elementos de una página web, definiendo su apariencia visual como colores, fuentes, márgenes, etc. Los navegadores web son los que interpretan el CSS, utilizando el lenguaje de marcado HTML para construir la estructura de la página y el CSS para aplicarle el estilo.

1.3- ¿A qué se llama regla de CSS?

En CSS, una regla es una instrucción que define cómo se verá un elemento HTML. Está compuesta por un selector, que indica qué elementos se van a estilizar, y una declaración, que contiene las propiedades de estilo y sus valores.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Una regla CSS se compone de un selector que muestra qué elementos HTML estilizar, seguido de un bloque de declaración que define las propiedades y valores que se aplicarán a esos elementos. La regla básica es selector { propiedad: valor; }.

1.5- ¿Qué es un selector en CSS?

En CSS, un selector es la primera parte de una regla que define qué elementos HTML deben ser estilizados con las propiedades y valores especificados en la regla. Es como una dirección que indica al navegador cuáles son los elementos a los que aplicar un estilo.

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Los selectores CSS son herramientas fundamentales para estilizar páginas web, permitiendo apuntar a elementos HTML específicos. El selector de ID (`#`) identifica un único elemento por su atributo `id`, mientras que el selector de clases (`.`) aplica estilos a múltiples elementos con el mismo atributo `class`. El selector universal (`*`) selecciona todos los elementos, el selector de etiquetas (como `p` o `div`) se dirige a todos los elementos de un tipo específico, y el selector descendente (espacio entre selectores) selecciona elementos que son descendientes de otro elemento.

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML (por ejemplo, lo recibe de la red).
  2. Convierte el HTML en un DOM (Modelo de objetos del documento). El DOM representa el documento en la memoria del ordenador. Lo explicaremos más detalladamente en la sección siguiente.
  3. Entonces, el navegador va a buscar la mayor parte de los recursos vinculados al documento HTML, como las imágenes y los videos incrustados... ¡y también el CSS vinculado! JavaScript aparece un poco más adelante en el proceso, pero no vamos a hablar de ello aún para evitar complicar las cosas.
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector. Por ejemplo, elemento, clase, ID, y así sucesivamente.
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página (esta etapa se llama pintura).

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Archivo CSS externo: El código se escribe en un archivo . css a parte. Bloque de estilos: El código se escribe en una etiqueta "style" en el documento HTML. Estilos en línea: El código se escribe en un atributo HTML style en una etiqueta.
